How It Works

  • Each timeline cell is colored based on total allocation percentage
  • Green - Person has capacity available
  • Orange - Person is mostly allocated
  • Red - Person is over-allocated (needs attention!)
  • Empty cells remain neutral colored

üìä Percentage-Based Allocation

Assign team members to projects with specific allocation percentages (1-100%).

Examples

  • 100% - Full-time on one project
  • 50% - Half-time (split between two projects)
  • 25% - Quarter-time (working on multiple projects)

How to Set Allocation

  1. Click "+ Assign to Project"
  2. Select person, project, and time period
  3. Enter allocation percentage (1-100)
  4. Click "Assign"
üí° Tip: You can assign the same person to multiple projects in one period. For example: 50% on Project A + 50% on Project B = 100% total allocation.

üñ±Ô∏è Drag and Drop

Move assignments between people and time periods with drag and drop!

How to Use

  1. Find the assignment card you want to move
  2. Click and hold the assignment card
  3. Drag it to the target cell (different person or time period)
  4. Release to drop - assignment moves instantly!

Visual Feedback

  • Assignment becomes semi-transparent while dragging
  • Target cell shows a dashed orange border
  • Cursor changes to indicate drag state
  • Drop updates both frontend and database
üí° Tip: You can drag assignments to any visible cell. Use the time navigation buttons to move between months, then drag assignments to future periods.
